# Praise Coffee ☕️

クールなパースペクティブをもつコーヒー屋さん [Praise Coffee](https://praise-coffee.com) のウェブサイトです。

>こんにちは、プレイズコーヒーです。変化がめまぐるしく多様化する時代の中でも、美味しいコーヒーを飲むときには、自由と幸せをココロで感じてもらいたいと、わたしたちは考えています。
